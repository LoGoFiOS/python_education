## Создание виртуального окружения

```bash
# создание папки venv с виртуальным окружением
python3 -m venv venv
# Переходим в виртуальную среду
source venv/bin/activate
# Установка пакетов
pip install -r requirements.txt
# выходим из виртуальной среды
deactivate
```
